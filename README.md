# MSc(Master of Science) Thesis , Machine Vision usage in motion control system
With many experimental and exciting experiences in SEECS Laboratory in real Multidisciplinary researches.... 
Machine vision systems are systems that use cameras to capture and process the signals that form an image, using computers or 
other signal processing devices to interpret and analyze the received signals. One of the applications of machine vision 
discussed in this study is the control of the position of a moving object or a moving part of an object (for example, a motor,
a rotor) by means of a camera (with image processing). In the image processing stage, the goal is to calculate the position and 
send the calculated position data (as a feedback) to the controller (digital control system) with processor implemented on a
microcontroller to control the position of the actuator. Also, in this study, the Lorentz linear actuator is used due to its 
simple structure, small dimensions, lack of lag, and lack of ripple in the output force and on the other hand, the operating - 
frequency of the system is one of the important parameters, because the execution time of the control process is directly 
related to the performance of the system, since the operating frequency is the same as the frame rate that set to 60 Hz -
(or 60 frame per second). In this research, also by following the movement of a linear electromagnetic actuator and calculating
the displacement and position at each frame of the image and calibrating the actual position, as position feedback, a position 
controller is implemented for this actuator to reduce or minimize the error signal (the difference between the actual position 
and the reference or desired position of the actuator) to zero, and then the results are examined. Also In this research, a 
digital control system or a digital controller designed with 2 zeros and 2 poles, which has an integrator to zeroing the Steady State Error(SSE) of the system, is used. 
Keywords: machine vision system, motion control system, feedback, image processing, position control, data exchange, signal-
exchange, digital controller.
